# 简单介绍
 根目录下：
 
     使用 <_NewPrj.bat> 创建新工程
     
     <_lib>文件夹下存放了一些编译用的批处理以及程序模板（暂时只有一个三段式状态机）
     
 工程目录下：
 
     使用 <_A2Z.bat> 编译&调用GTKWave显示仿真波形
     
     使用 <_WaveDisplay.bat> 调用GTKWave显示已存在的波形文件
     
     <*.v>为verilog程序文件
     
     <*_tb.v>为Testbenh
